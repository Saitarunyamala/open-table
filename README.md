# OpenTable Clone (ASP.NET Core MVC)

An OpenTable-inspired web application built using ASP.NET Core MVC.
This project follows a clean MVC architecture with Controllers, Models, Views, Areas, and static assets.

## Tech Stack
- ASP.NET Core MVC (.NET)
- C#
- Razor Views
- HTML, CSS
- JavaScript

## Project Structure
open-table
- Areas
- Controllers
- Models
- Views
- wwwroot
- Program.cs
- appsettings.json

## Project Purpose
This project is built as a learning and portfolio application to demonstrate MVC architecture, routing, and scalable web application design using ASP.NET Core.

## Features
- MVC-based architecture
- Area-based routing support
- Clean separation of concerns
- Responsive layout structure

## Planned Features
- User authentication
- Restaurant listings
- Restaurant details page
- Table reservation workflow
- Admin dashboard for management

## Configuration
Application settings are stored in appsettings.json.
Sensitive data should not be committed to the repository.

Project is under active development.
